
# Portal de Conscientização Digital

Projeto prático desenvolvido para o Plano de Recuperação Disciplinar da disciplina de Educação Digital e Computação: Programação e IA.

## Objetivo do Tema
O site atua como um portal comunitário focado em debater a Cidadania Digital, explicando os impactos causados por mídias manipuladas (Deepfakes) e os perigos da desinformação na sociedade moderna. O objetivo é educar o usuário de forma interativa através de conteúdos informativos e um quiz.

##  Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo (uso de tags como `header`, `nav`, `main`, `section`, `footer` e elementos de formulário).
- **CSS3:** Estilização baseada na paleta verde-água/ciano, uso de Flexbox para alinhamento dos cards e navegação, além de regras de responsividade para celulares.
- **JavaScript Vanilla:** Lógica para manipulação dinâmica do DOM, controle das perguntas do quiz, exibição de dicas e alternância do Modo Escuro.

##  Uso de Inteligência Artificial e Documentação de Prompts
Seguindo os critérios de codesign ético estabelecidos no plano de recuperação, a Inteligência Artificial foi utilizada como ferramenta de suporte no desenvolvimento. Abaixo estão listados os principais comandos (prompts) utilizados:

1. *"Gere uma estrutura de dados em formato de array de objetos Javascript contendo perguntas sobre deepfakes e segurança para um quiz interativo."*
2. *"Como estruturar uma função em JavaScript que mude a classe do body para alternar entre modo claro e escuro clicando em um botão?"*
3. *"Como criar um efeito de transição suave em cards de conteúdo usando CSS com hover e transform?"*
