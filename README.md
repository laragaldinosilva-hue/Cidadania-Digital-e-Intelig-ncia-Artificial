# Cidadania-Digital-e-Intelig-ncia-Artificial
# Portal de Conscientização Digital 

Projeto prático desenvolvido para o Plano de Recuperação Disciplinar da disciplina de Educação Digital e Computação: Programação e IA.

##  Objetivo do Tema
O site visa atuar como um portal comunitário focado em debater a **Cidadania Digital e os impactos causados por mídias manipuladas (Deepfakes)** e pela desinformação gerada por Inteligências Artificial automatizadas.

## Tecnologias Utilizadas
- HTML5 (Estruturação Semântica)
- CSS3 (Estilização Avançada com Flexbox e Responsividade)
- JavaScript Vanilla (Manipulação Dinâmica do DOM)

##  Uso de Inteligência Artificial e Documentação de Prompts
De acordo com os critérios éticos estabelecidos pelo plano, os seguintes prompts foram aplicados para auxílio no codesign do projeto:
1. *"Gere uma estrutura de dados em formato de array de objetos Javascript contendo perguntas sobre deepfakes e segurança para um quiz iterativo."*
2. *"Como criar um efeito de transição suave em cards CSS utilizando hover e transform?"*
